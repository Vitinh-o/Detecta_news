Detecta News

O porquê deste projeto

Este projeto foi desenvolvido com o objetivo de criar uma ferramenta prática e acessível para identificação de fake news, unindo tecnologia e responsabilidade social. Com o avanço da desinformação nas redes sociais, cresce a necessidade de soluções tecnológicas que ajudem o público a verificar rapidamente a veracidade de conteúdos compartilhados. A aplicação combina tecnologias modernas como HTML, Bootstrap 5, CSS e JavaScript no frontend com um backend em Python utilizando o framework FastAPI. A Inteligência Artificial integrada ao sistema, baseada no modelo spaCy TextCatBow v3, é capaz de analisar textos e indicar a probabilidade de serem falsos ou verdadeiros, tornando a ferramenta uma aliada no combate à desinformação.

Metodologia

A metodologia adotada no projeto se baseia em duas etapas principais de verificação. Primeiramente, ao receber o título da notícia, a aplicação realiza uma busca automatizada em sites de checagem confiáveis utilizando a API Google Search, verificando se aquela informação já foi analisada e classificada como verdadeira ou falsa por fontes oficiais.

Caso não haja resultados suficientes, a notícia é então processada por nossa Inteligência Artificial. O texto passa por um pré-processamento (tokenização, remoção de pontuação, conversão para minúsculas e possível lematização) e é transformado em uma representação numérica usando a técnica Bag-of-Words. Com base nessa vetorização, o modelo classifica a notícia de acordo com padrões aprendidos durante o treinamento com o corpus Fake.br, indicando a probabilidade de ela ser falsa.
